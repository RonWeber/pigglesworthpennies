What is PigglesworthPennies?
----------------

PigglesworthPennies is an experimental digital currency that enables instant payments to
anyone, anywhere in the world. PigglesworthPennies uses peer-to-peer technology to operate
with no central authority: managing transactions and issuing money are carried
out collectively by the network. PigglesworthPennies Core is the name of open source
software which enables the use of this currency.

If your want more information, or an immediately useable, binary version of
the PigglesworthPennies Core software, you're kinda SOL.
